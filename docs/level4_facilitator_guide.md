## ANSWER KEY WITH FRAMEWORK INTEGRATION

### Q1: The Billion-Dollar iPad Disaster
**Correct: B**
- **PICRAT:** Design learning activities BEFORE purchasing
- **Teacher PD:** Extensive training on pedagogical integration
- **4Ds:** Train teachers on AI/tech delegation boundaries
- **UDL:** Ensure curriculum works for ALL learners

### Q2: The Interactive Whiteboard Graveyard
**Correct: C**
- **PICRAT:** Diagnose PR → Prescribe IA/IT (students touch the board)
- **Bloom's:** Use IWB for Apply/Analyze, not just Remember
- **UDL (HOW):** Kinesthetic expression through board interaction
- **Peer Coaching:** Ongoing support, not one-time training

### Q3: The Copy-Paste Catastrophe
**Correct: B**
- **4Ds D1:** Define AI CAN vs. student MUST
- **4Ds D3:** Evaluate and edit AI suggestions
- **4Ds D4:** Process portfolio showing drafts and prompts
- **Bloom's:** Add Evaluate level (critique AI output)
- **UDL (HOW):** Multiple output options

### Q4: The Kahoot! Addiction
**Correct: B**
- **Bloom's Audit:** Move beyond Remember level
- **PICRAT:** IR → IA/IT (formative assessment that changes instruction)
- **UDL (WHY):** Diversify engagement beyond competition
- **4Ds:** Use AI for varied question types

### Q5: The Language Lab Ghost Town
**Correct: C**
- **PICRAT Design:** Define target cells before purchase
- **Teacher Sustainability:** Train multiple teachers + documentation
- **Budget Planning:** Include 5-year maintenance in original budget
- **UDL Integration:** Multiple purposes for lab activities
- **Assessment Alignment:** Connect to grades for mandatory usage

### Q6: The One-Device Classroom War
**Correct: B**
- **UDL (WHAT):** Project onto wall for visibility
- **UDL (WHY):** Fair rotation schedule
- **UDL (HOW):** BYOD for parallel activities
- **PICRAT Station Rotation:** 4 stations, 15-min rotations
- **Bloom's:** Tablet for Create/Analyze; other stations for Apply
- **4Ds:** Research on phones, curate on tablet

### Q7: The Zoom Zombie Apocalypse
**Correct: C**
- **PICRAT:** 10-min chunks with interaction (PA → IA)
- **Bloom's Chunking:** Each chunk targets one cognitive level
- **UDL (WHAT):** Recording + transcript + visual summary
- **UDL (WHY):** Varied activities for attention
- **UDL (HOW):** Multiple participation modes
- **4Ds:** AI for summaries and follow-up questions

## DISCUSSION PROMPTS FOR FACILITATOR

After each scenario, you can ask:

1. **"Which equation does this scenario illustrate?"**
   - Help participants connect the disaster to a memorable formula

2. **"What would a WRONG framework application look like?"**
   - Discuss why the distractors are inadequate

3. **"Has anyone experienced something similar?"**
   - Connect to participants' real experiences

4. **"What's the ONE thing that could have prevented this?"**
   - Focus on root cause, not symptoms
